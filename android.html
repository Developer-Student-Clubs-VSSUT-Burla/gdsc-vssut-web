About the Program

Android study jams is a community driven program to help students learn to build Android apps in the Kotlin programming language and earn badges/certificates along the way.Participants completing this course and projects receive a certificate from Google's swag. Participants also get discounts on the Android nanodegree if they choose to continue their Android Journey.

The first track is for participants new to programming, that deals with android basics in Kotlin . In the first track you will learn about basic programming concepts of. The unit one is about Kotlin basics for Android and second unit is about layouts .Unit one has four pathways They are - Introduction to Kotlin, Create your first Android app, Build a basic layout and add a button to an app.In the second unit you will learn to get a user input and create UI components such as editable text fields , radio button , switches and learn to change the theme and launcher icon for your app.

The second track is for participants having prior programming experience. Here, you have to go through a series of exercises to build your first interactive app , create different layouts , learn about working of multi screen navigation, activity and fragment lifecycle. You will further learn about architecture components like Viewmode and Livedata and create a database using the room library.

After completion of each unit you will earn digital badges which will be displayed in the developer's profile. You will also get a chance to showcase your app on official Google Platforms!

It is free of cost. All it requires is dedication and exciting goodies await you.
